# Simplex

**Simplex** is the zero-effort media renamer for Plex. Clean up your messy TV show and movie filenames in seconds with intelligent metadata fetching and a beautiful, high-performance interface.

## ‚ú® Key Features

- **Drag & Drop**: Simply drop a folder of messy files and let Simplex do the heavy lifting.
- **Intelligent Parsing**: Sophisticated regex engine that understands complex release group tags, quality markers, and varied episode formats.
- **Automated Metadata**: Leverages TVMaze and YTS APIs to automatically fetch correct show titles, release years, and episode names.
- **Plex-Standard Formatting**: Automatically organizes files into the standard `Show Name (Year)/Season XX/Episode Name - SXXEXX` structure.
- **Turbo Mode**: Process entire seasons or movie collections with a single click.
- **Built for Speed**: A high-performance tool designed for power users.
